# Snake Game

## Overview
This is a simple Snake game developed using Python and the Pygame library. In this game, players control a snake that grows by eating food. The objective is to grow the snake as long as possible without colliding with its own body or the game boundaries.

## Game Features
- Simple controls: Use the arrow keys to move the snake.
- Growth mechanism: The snake grows by one segment each time it eats food.
- Game over condition: The game ends if the snake collides with its own body or the game window boundaries.

## Running the Game
To run this game, ensure you have Python and the Pygame library installed in your environment. Then, follow these steps:

1. Clone or download this project to your local machine.
2. Open a terminal or command prompt, navigate to the directory containing the game files.
3. Run the following command to start the game:
   ```bash
   python game.py
